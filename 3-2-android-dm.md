
# Android Device Manager

## Overview
Android Device Manager is a system in QE that we can get control of an android device to QE. we can use it's sensors, cameras, microphone etc and send these data to QE blocks and use them using [Nexus](https://qepal.com/docs/4-nexus.md).

---

## Getting Started

At the first time we need to install [Worker Android APK File](https://cdn.qepal.com/uploads/worker.api) and [API Android APK File](https://cdn.qepal.com/uploads/api.api) and then from [QE Onlines List](https://qepal.com/fa/onlines) click on Add and enter the MAC address that is showed on the Android Device, and it's done.
